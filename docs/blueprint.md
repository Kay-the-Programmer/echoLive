# **App Name**: StreamVerse

## Core Features:

- Live Streaming: Real-time video and audio broadcasting for user engagement.
- Party Rooms: Virtual rooms with tiered seating (4, 9, 16, 21, 25, 35 seats) and special crown seat rules to provide engagement.
- PK Battles: Heated Player-vs-Player competition system
- Coin Economy: In-app currency system to facilitate transactions, gifting, and VIP upgrades with bonus rewards.
- Generative Gift Recommendation: An AI tool to suggest relevant gifts based on streamer profile and viewer preferences to maximize engagement
- Leveling System: Gamified leveling system (Livestream Level and Wealth Level) driven by points, coins, and engagement milestones to encourage participation.
- Withdrawal System: A system for users to convert earned points to USDT and withdraw via crypto platforms.

## Style Guidelines:

- Primary color: Electric Purple (#BE29EC) to create a futuristic and energetic atmosphere.
- Background color: Dark Gray (#232323), offering contrast and emphasizing other colors. 
- Accent color: Hot Pink (#EC298F) for interactive elements and highlights.
- Body font: 'Inter', a sans-serif font, known for a neutral, objective style
- Headline font: 'Space Grotesk', a sans-serif font with a futuristic style suitable for titles; for longer text use 'Inter'.
- Use neon-style icons to represent different features and actions.
- Subtle animations for level transitions and coin rewards.
